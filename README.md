# 🤖 Multi-Agent AI System

A Multi-Agent AI application built using **LangGraph**, **LangChain**, and **Python** that coordinates multiple intelligent agents to solve user tasks through a structured workflow.

---

## 📌 Overview

This project demonstrates a multi-agent architecture where specialized AI agents collaborate to complete complex tasks. Each agent has a specific responsibility, improving modularity, scalability, and maintainability.

---

## ✨ Features

- Multi-Agent Workflow
- LangGraph State Management
- LangChain Integration
- Tool Calling Support
- Modular Agent Design
- Environment Variable Configuration
- Extensible Architecture
- Easy Integration with LLM APIs

---

## 🛠️ Tech Stack

- Python
- LangGraph
- LangChain
- OpenAI / Groq API
- python-dotenv

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sripada-Sudheer28/Muti-Agent-System.git
```

### Move into Project

```bash
cd Muti-Agent-System
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows

```bash
.venv\Scripts\activate
```

Linux / macOS

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

Example:

```env
OPENAI_API_KEY=your_api_key
GROQ_API_KEY=your_api_key
```

> Never upload your actual `.env` file to GitHub.
